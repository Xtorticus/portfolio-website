Hello World! Welcome to Abe Talks Font-End.This website got it start back in March on my drive home from work. The Idea struck me to make and host my own website. With that idea and about $150.00 abrahamzimmerman.com was launched on a Wordpress template.The goal of this website is to outline my front-end development skills and to learn about web development in the wild.

As a press operator for the entirety of my career learning on the fly and producing results on deadline are second nature. This blog "Abe Talks Font-End" will be the outline and discussion for everything related to the development of this website and various other projects.

I enjoy working with HTML, CSS, JavaScript, JQuery, and Phython. I currently use two different computers for development. My main computer is a tower located at home running Windows 10. I also have Adobe Creative Cloud in my main system to make and edit graphics. My laptop is dual booting Ubuntu and Windows 10. However, I hardly use the Windows partition on my laptop due to the accessibility of Ubuntu.To make transitioning between machines I keep all my work update on Github.

Current Setup

Currently my text editor of choice in Sublime Text 3. I have used Notepad ++ in the past and have thought about checking out Visual Studio Code. I use browser-sync to keep changes current while I am developing. Also, I use git to push all work to to my github profile. Github is great when working on multiple platforms. It allows me to keep everything current without much chance of error. With Sublime, Git, and Browser Sync I have found myself a comfortable setup that is easy to maintain.

In the future I will update you on the changes to my setup as they happen. If I move to a different code editor or begin working with a CMS like WordPress. The idea is not to keep anything static and to be constantly be exploring different methodologies.

I will keep this first blog short and sweet. Thanks for stopping by, I hope to see you again.
